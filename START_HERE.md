## 🚀 Quick Start

### 1. Build the Project
```bash
cd ~/Desktop/spoof/CameraSpoof
chmod +x gradlew
./gradlew assembleDebug
```

### 2. Install on Device
```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```

### 3. Activate in LSPosed
1. Open LSPosed Manager
2. Enable CameraSpoof module
3. Add System Framework to scope
4. Reboot device

### 4. Configure and Use
1. Open CameraSpoof app
2. Select target app
3. Choose image
4. Enable spoofing
5. Test in target app
