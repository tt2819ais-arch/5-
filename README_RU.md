# CameraSpoof

Production-ready Android LSPosed/Xposed модуль для подмены изображений камеры в выбранных приложениях.

## Возможности

- 📱 Подмена изображений камеры в любом приложении
- 🎯 Настройка для каждого приложения отдельно
- 📷 Поддержка фронтальной, основной или всех камер
- 🖼️ Предпросмотр изображений
- 📊 Система логирования
- 🌙 Material 3 Dark Theme
- 💾 Сохранение настроек между перезагрузками
- 🔍 Поиск приложений

## Требования

- Android 10 - 15 (API 29-35)
- LSPosed или Xposed Framework
- Magisk/Zygisk (рекомендуется)
- Root доступ

## Установка

1. Установите LSPosed/Xposed Framework
2. Скачайте последний APK из [Releases](../../releases)
3. Установите APK
4. Активируйте модуль в LSPosed Manager
5. Выберите целевые приложения в области действия LSPosed
6. Перезагрузите устройство

## Использование

1. Откройте приложение CameraSpoof
2. Найдите и выберите приложение для подмены
3. Включите переключатель
4. Выберите изображение из галереи
5. Выберите режим подмены (Фронтальная/Основная/Все камеры)
6. Откройте целевое приложение и используйте камеру

## Сборка

### Локальная сборка

```bash
git clone https://github.com/yourusername/CameraSpoof.git
cd CameraSpoof
./gradlew assembleDebug
```

### GitHub Actions

Проект включает автоматический CI/CD pipeline для сборки APK при каждом push.

## Архитектура

- **MVVM** паттерн архитектуры
- **Jetpack Compose** для UI
- **Room** для локальной базы данных
- **Coroutines** и **Flow** для асинхронных операций
- **Repository** паттерн для управления данными

## Поддерживаемые Camera API

- Camera2 API
- Camera1 API (legacy)

## Логи

Приложение включает систему логирования:
- Логи успеха/неудачи хуков
- Определение активных пакетов
- Определение Camera API
- Логи внедрения изображений
- Экспорт логов в текстовый файл

## Безопасность

- Не ломает системную камеру
- Работает только в выбранных приложениях
- Корректная обработка ошибок
- Нет риска bootloop
- Fallback поведение при ошибках
